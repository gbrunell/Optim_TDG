# Optim_TDG
Optim Test Data Generator

Written by G Brunell using Java and Optim Command Line Interface

Requires the use of a java compiler that will vary depending on your platform and architecture (windows /64 bit or windows /32

I built and have used this successfully with an older Java JDK jdk-6u20-windows-i586  which you will need to install so that you can compile the java program called generate
